Apache Velocity
-------------------------------------------------------------------------------
https://velocity.apache.org/

Velocity is a Java-based template engine. It permits anyone to use a simple yet
powerful template language to reference objects defined in Java code.

Velocity's capabilities reach well beyond the realm of the web, it can be used
to generate SQL, PostScript and XML from templates.

> Getting Started
    https://velocity.apache.org/engine/2.2/getting-started.html
